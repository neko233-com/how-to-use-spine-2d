# Peer Spine Project Study

This directory is for deconstructing public Spine projects and recording rebuild
notes. Do not copy proprietary project files or third-party commercial assets
into this repository.

## Sources To Study

- Spine official example projects: https://esotericsoftware.com/spine-examples
- Spine runtimes examples: https://github.com/EsotericSoftware/spine-runtimes
- Super Spineboy: https://github.com/EsotericSoftware/spine-superspineboy

## Output Format

For each peer project, create one note with:

1. public source link,
2. what the project demonstrates,
3. skeleton and slot observations,
4. runtime/export observations,
5. a smaller rebuild assignment using local assets.
