# 界面概览

## 主界面布局

Spine 的界面由以下几个主要区域组成：

```
┌─────────────────────────────────────────────────────────┐
│  菜单栏 (Menu Bar)                                       │
├──────────────┬──────────────────────────┬───────────────┤
│  工具栏      │      视口 (Viewport)      │   属性面板    │
│  (Tools)     │                          │  (Properties) │
│              │                          │               │
│              │                          │               │
│              │                          │               │
├──────────────┴──────────────────────────┴───────────────┤
│  大纲视图 (Outliner)                                     │
├─────────────────────────────────────────────────────────┤
│  时间轴 (Timeline)                                       │
└─────────────────────────────────────────────────────────┘
```

## 菜单栏

### File（文件）

- **New**：新建项目
- **Open**：打开项目
- **Save**：保存项目
- **Export**：导出动画

### Edit（编辑）

- **Undo**：撤销
- **Redo**：重做
- **Preferences**：首选项设置

### View（视图）

- **Zoom In/Out**：缩放视图
- **Reset View**：重置视图
- **Show Grid**：显示网格

### Spine

- **New Skeleton**：新建骨架
- **New Animation**：新建动画
- **New Skin**：新建皮肤

## 工具栏

### 选择工具

- **Select**：选择骨骼、图片等对象
- **Move**：移动对象
- **Rotate**：旋转对象
- **Scale**：缩放对象

### 骨骼工具

- **Create Bone**：创建骨骼
- **Create IK**：创建 IK 约束

### 编辑工具

- **Create Mesh**：创建网格
- **Edit Mesh**：编辑网格顶点

## 视口 (Viewport)

视口是您查看和编辑骨架的主要区域。

### 视图控制

- **鼠标滚轮**：缩放视图
- **鼠标中键拖动**：平移视图
- **鼠标右键拖动**：旋转视图（3D 模式）

### 视图模式

- **Pose**：姿势模式，用于调整骨骼位置
- **Animate**：动画模式，用于创建关键帧
- **Setup**：设置模式，用于初始骨架搭建

## 大纲视图 (Outliner)

大纲视图显示骨架的层级结构。

### 骨骼层级

- 父骨骼在上，子骨骼在下
- 拖动可以调整层级关系
- 右键可以添加、删除、重命名

### 图片列表

- 显示所有导入的图片
- 可以将图片绑定到骨骼

## 属性面板 (Properties)

属性面板显示选中对象的详细属性。

### 骨骼属性

- **Position**：位置 (X, Y)
- **Rotation**：旋转角度
- **Scale**：缩放比例
- **Length**：骨骼长度

### 图片属性

- **Position**：位置
- **Rotation**：旋转
- **Scale**：缩放
- **Color**：颜色和透明度

## 时间轴 (Timeline)

时间轴是动画编辑的核心区域。

### 时间轴控制

- **播放/暂停**：预览动画
- **时间滑块**：移动到特定帧
- **关键帧按钮**：添加/删除关键帧

### 关键帧

- 菱形图标表示关键帧
- 可以选择、移动、复制关键帧
- 支持贝塞尔曲线调整
