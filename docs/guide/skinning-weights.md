# 蒙皮与权重

## 什么是蒙皮？

蒙皮（Skinning）是将网格绑定到骨骼系统的过程，使网格能够跟随骨骼运动并产生变形。

## 蒙皮类型

### 线性混合蒙皮 (Linear Blend Skinning)

- 最常用的蒙皮方式
- 每个顶点受多个骨骼影响
- 通过权重控制影响程度

### 线性混合公式

```
final_position = Σ(weight_i * bone_i_transform * original_position)
```

## 权重绘制

### 什么是权重？

权重定义了网格顶点受骨骼影响的程度。

### 权重值范围

- **0.0**：完全不受影响
- **1.0**：完全受影响
- **0.5**：50% 影响

### 绘制权重步骤

1. 选中网格
2. 选择 **Weight** 工具
3. 选择要绘制的骨骼
4. 在顶点上拖动绘制权重

### 权重绘制模式

| 模式 | 说明 |
|------|------|
| Add | 增加权重 |
| Replace | 替换权重 |
| Subtract | 减少权重 |
| Smooth | 平滑权重 |

## 权重绘制技巧

### 使用平滑工具

1. 选择 **Smooth** 模式
2. 在权重突变区域拖动
3. 平滑过渡权重值

### 避免权重突变

- 权重变化要平滑过渡
- 避免相邻顶点权重差异过大
- 使用平滑工具修正

### 测试变形效果

1. 在不同姿势下测试网格
2. 检查是否有拉伸或撕裂
3. 调整权重修正问题

## 自动权重

### 使用自动权重

1. 选中网格
2. 点击 **Auto Weight** 按钮
3. Spine 会自动计算权重
4. 手动调整不准确的部分

### 自动权重的局限

- 可能不够精确
- 复杂网格需要手动调整
- 建议在自动权重基础上手动优化

## 权重优化

### 减少骨骼影响数

- 每个顶点最多受 4 个骨骼影响
- 删除影响很小的权重
- 优化性能

### 权重归一化

- 确保所有权重之和为 1.0
- 使用 **Normalize** 功能自动归一化
- 避免权重异常

### 权重镜像

- 使用 **Mirror** 功能镜像权重
- 保持左右对称
- 节省绘制时间

## 蒙皮工作流程

### 完整流程

1. **创建骨骼**：建立骨架结构
2. **导入图片**：导入角色图片
3. **创建网格**：将图片转为网格
4. **绑定网格**：将网格绑定到骨骼
5. **绘制权重**：调整权重值
6. **测试变形**：在不同姿势下测试

### 最佳实践

- 先创建完整的骨架
- 再导入图片并创建网格
- 最后绘制权重
- 边绘制边测试

## 常见问题

### Q: 权重绘制后有拉伸怎么办？

A: 调整权重值，确保过渡平滑，使用平滑工具修正。

### Q: 如何快速绘制对称的权重？

A: 使用镜像功能，先绘制一侧，再镜像到另一侧。

### Q: 每个顶点最多受几个骨骼影响？

A: 建议最多 4 个，过多会影响性能和效果。
