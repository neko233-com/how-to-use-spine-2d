# Spine 简介

## 什么是 Spine？

Spine 是一款专业的 2D 骨骼动画编辑器，由 Esoteric Software 开发。它专门用于创建高质量的 2D 游戏动画，被广泛应用于游戏开发行业。

## 为什么选择 Spine？

### 优势

- **专业级工具**：专为游戏动画设计，功能强大且高效
- **骨骼动画**：基于骨骼系统，动画文件小，运行效率高
- **网格变形**：支持网格和权重，实现流畅的变形动画
- **运行时支持**：支持 Unity、Unreal、Cocos2d 等主流游戏引擎
- **活跃社区**：拥有庞大的用户社区和丰富的学习资源

### 适用场景

- 2D 游戏角色动画
- UI 动画效果
- 过场动画
- 动态插图
