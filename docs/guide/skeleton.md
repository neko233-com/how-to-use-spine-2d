# 骨骼系统

## 什么是骨骼？

骨骼是 Spine 动画系统的基础。它定义了角色的结构和运动方式。

### 骨骼层级

骨骼形成父子层级关系：

```
root
├── hip
│   ├── left-leg
│   │   ├── left-thigh
│   │   └── left-calf
│   └── right-leg
│       ├── right-thigh
│       └── right-calf
├── spine
│   ├── chest
│   │   ├── left-arm
│   │   └── right-arm
│   └── head
```

### 父子关系

- 子骨骼跟随父骨骼运动
- 移动父骨骼会影响所有子骨骼
- 子骨骼可以有自己独立的运动

## 创建骨骼

### 步骤

1. 选择 **Create Bone** 工具
2. 在视口中点击确定骨骼起点
3. 拖动鼠标确定骨骼方向和长度
4. 点击完成创建

### 技巧

- 从角色的根部开始创建（如臀部）
- 按照身体自然结构创建层级
- 使用参考图片辅助定位

## 编辑骨骼

### 选择骨骼

- 点击骨骼选中
- 按住 Shift 多选
- 在大纲视图中点击选中

### 移动骨骼

1. 选择 **Move** 工具
2. 拖动骨骼到新位置
3. 子骨骼会跟随移动

### 旋转骨骼

1. 选择 **Rotate** 工具
2. 拖动旋转骨骼
3. 子骨骼会跟随旋转

### 调整骨骼长度

1. 选中骨骼
2. 在属性面板中修改 **Length** 值
3. 或拖动骨骼末端控制点

## 骨骼属性

### 基本属性

| 属性 | 说明 |
|------|------|
| Name | 骨骼名称 |
| Parent | 父骨骼 |
| Position | 位置 (X, Y) |
| Rotation | 旋转角度 |
| Scale | 缩放 (X, Y) |
| Length | 骨骼长度 |

### 高级属性

| 属性 | 说明 |
|------|------|
| Shear | 剪切变形 |
| Transform | 变换模式 |
| Color | 骨骼颜色 |

## 骨骼层级管理

### 重新排序

- 在大纲视图中拖动骨骼
- 调整父子关系
- 影响动画继承

### 复制骨骼

1. 选中骨骼
2. Ctrl+D 复制
3. 调整复制骨骼位置

### 删除骨骼

1. 选中骨骼
2. 按 Delete 键
3. 确认删除

::: warning 注意
删除父骨骼会同时删除所有子骨骼。
:::

## IK 骨骼

IK（反向运动学）骨骼可以简化动画制作。

### 创建 IK 骨骼

1. 选择 **Create IK** 工具
2. 点击 IK 链的起始骨骼
3. 点击 IK 链的结束骨骼
4. 设置 IK 约束

### IK 优势

- 移动末端自动计算中间骨骼角度
- 适合腿部、手臂动画
- 减少关键帧数量

## 常见问题

### Q: 骨骼创建后如何调整位置？

A: 使用 Move 工具拖动骨骼，或在属性面板中修改 Position 值。

### Q: 如何让子骨骼不跟随父骨骼旋转？

A: 在属性面板中设置 Transform 为 **Only Translation**。

### Q: 骨骼命名有什么规范？

A: 建议使用有意义的英文名称，如 `left-arm`、`right-leg`。
