# 安装与配置

## 系统要求

### 最低配置

- **操作系统**：Windows 10/11、macOS 10.15+、Linux
- **处理器**：64 位处理器
- **内存**：4 GB RAM
- **显卡**：支持 OpenGL 3.3
- **硬盘**：500 MB 可用空间

### 推荐配置

- **内存**：8 GB RAM 或更多
- **显卡**：独立显卡，支持 OpenGL 4.5
- **显示器**：1920x1080 或更高分辨率

## 下载 Spine

1. 访问 [Spine 官网](https://esotericsoftware.com/spine-professional)
2. 选择适合您操作系统的版本
3. 点击下载按钮

::: warning 注意
Spine 是付费软件，但提供 30 天免费试用期。
:::

## 安装步骤

### Windows

1. 运行下载的安装程序
2. 按照安装向导提示完成安装
3. 启动 Spine

### macOS

1. 打开下载的 .dmg 文件
2. 将 Spine 拖到 Applications 文件夹
3. 从 Applications 启动 Spine

### Linux

1. 解压下载的压缩包
2. 运行 `spine` 可执行文件
3. 或使用包管理器安装（如果可用）

## 激活许可证

1. 启动 Spine
2. 点击 **Help** > **Enter License Key**
3. 输入您的许可证密钥
4. 点击 **Activate**

## 配置首选项

### 界面语言

1. 点击 **Edit** > **Preferences**
2. 在 **Language** 下拉菜单中选择语言
3. 重启 Spine 使设置生效

### 快捷键设置

Spine 支持自定义快捷键：

1. 点击 **Edit** > **Preferences** > **Shortcuts**
2. 选择要修改的命令
3. 按下新的快捷键组合

### 自动保存

建议开启自动保存功能：

1. 点击 **Edit** > **Preferences**
2. 设置 **Auto Save** 间隔（建议 5 分钟）
3. 设置自动保存文件数量

## 常见问题

### Q: Spine 无法启动怎么办？

A: 检查显卡驱动是否更新，确保支持 OpenGL 3.3 或更高版本。

### Q: 如何更新 Spine？

A: Spine 会自动检查更新，或在 **Help** > **Check for Updates** 手动检查。

### Q: 试用期结束后怎么办？

A: 购买许可证或联系 Esoteric Software 获取延期。
