# 实战案例

## 案例概述

本章通过完整的游戏角色动画案例，综合运用前面所学的知识。

## 案例 1：2D 角色待机动画

### 目标

创建一个简单的 2D 角色待机动画。

### 步骤

#### 1. 准备素材

- 角色身体各部分图片（头、身体、手臂、腿部）
- 确保图片透明背景
- 图片命名规范

#### 2. 创建骨架

```
root
├── hip
│   ├── left-leg
│   │   ├── left-thigh
│   │   └── left-calf
│   └── right-leg
│       ├── right-thigh
│       └── right-calf
├── spine
│   ├── chest
│   │   ├── left-arm
│   │   │   ├── left-upper-arm
│   │   │   └── left-forearm
│   │   └── right-arm
│   │       ├── right-upper-arm
│   │       └── right-forearm
│   └── head
```

#### 3. 导入图片并绑定

1. 导入所有身体部分图片
2. 将图片绑定到对应骨骼
3. 调整图片位置和大小

#### 4. 创建网格（可选）

1. 选中需要变形的图片
2. 创建网格
3. 绘制权重

#### 5. 制作待机动画

1. 新建动画 `idle`
2. 设置初始姿势
3. 添加呼吸动画（身体轻微上下）
4. 添加手臂轻微摆动
5. 调整动画曲线

#### 6. 导出测试

1. 导出 JSON 格式
2. 导出纹理图集
3. 在游戏引擎中测试

### 关键技巧

- 待机动画要 subtle（微妙）
- 使用贝塞尔曲线让动画更自然
- 循环播放时注意首尾衔接

## 案例 2：2D 角色行走动画

### 目标

创建一个流畅的行走循环动画。

### 步骤

#### 1. 分析行走动作

- 腿部交替运动
- 手臂自然摆动
- 身体轻微上下起伏
- 头部保持稳定

#### 2. 创建动画

1. 新建动画 `walk`
2. 设置关键帧姿势

#### 3. 关键帧设置

| 帧 | 腿部 | 手臂 | 身体 |
|----|------|------|------|
| 0 | 左腿前，右腿后 | 右臂前，左臂后 | 最高点 |
| 12 | 双腿交叉 | 手臂经过身体 | 最低点 |
| 24 | 右腿前，左腿后 | 左臂前，右臂后 | 最高点 |

#### 4. 调整动画曲线

- 腿部运动使用平滑曲线
- 身体起伏使用正弦曲线
- 手臂摆动使用贝塞尔曲线

#### 5. 测试循环

- 确保第一帧和最后一帧相同
- 循环播放测试流畅度
- 调整不自然的部分

### 关键技巧

- 行走动画要参考真实动作
- 注意重心转移
- 手臂摆动要自然

## 案例 3：2D 角色攻击动画

### 目标

创建一个有力量感的攻击动画。

### 步骤

#### 1. 分析攻击动作

- 蓄力阶段
- 攻击阶段
- 收招阶段

#### 2. 创建动画

1. 新建动画 `attack`
2. 设置关键帧姿势

#### 3. 关键帧设置

| 帧 | 动作 | 说明 |
|----|------|------|
| 0 | 蓄力 | 手臂后拉，身体扭转 |
| 6 | 攻击 | 手臂前挥，身体前倾 |
| 12 | 命中 | 手臂到达目标位置 |
| 18 | 收招 | 手臂回收，身体恢复 |

#### 4. 添加事件

1. 创建 `attack-start` 事件
2. 创建 `attack-hit` 事件
3. 创建 `attack-end` 事件
4. 在对应帧添加事件

#### 5. 添加音效

1. 创建音频事件
2. 设置音效文件名
3. 在攻击帧添加音效

### 关键技巧

- 攻击动画要有力量感
- 使用快慢对比增强表现力
- 事件和音效要精确同步

## 案例 4：2D 角色跳跃动画

### 目标

创建一个流畅的跳跃动画。

### 步骤

#### 1. 分析跳跃动作

- 起跳准备
- 跳跃上升
- 空中姿态
- 下落
- 落地缓冲

#### 2. 创建动画

1. 新建动画 `jump`
2. 设置关键帧姿势

#### 3. 关键帧设置

| 帧 | 动作 | 说明 |
|----|------|------|
| 0 | 准备 | 蹲下蓄力 |
| 4 | 起跳 | 腿部伸直，身体上升 |
| 12 | 空中 | 身体展开，腿部弯曲 |
| 20 | 下落 | 身体收缩，准备落地 |
| 24 | 落地 | 腿部缓冲，身体下沉 |

#### 4. 添加事件

1. 创建 `jump-start` 事件
2. 创建 `jump-peak` 事件
3. 创建 `jump-land` 事件

### 关键技巧

- 跳跃动画要注意重力感
- 起跳和落地要有缓冲
- 空中姿态要自然

## 综合项目：完整角色动画集

### 目标

创建一个完整的角色动画集。

### 动画列表

| 动画 | 说明 | 循环 |
|------|------|------|
| idle | 待机 | 是 |
| walk | 行走 | 是 |
| run | 跑步 | 是 |
| jump | 跳跃 | 否 |
| attack | 攻击 | 否 |
| hurt | 受伤 | 否 |
| death | 死亡 | 否 |

### 工作流程

1. **创建骨架**：建立完整的角色骨架
2. **导入图片**：导入所有身体部分图片
3. **绑定图片**：将图片绑定到骨骼
4. **创建网格**：为需要变形的部分创建网格
5. **绘制权重**：调整权重值
6. **制作动画**：逐个制作动画
7. **添加事件**：为动画添加事件
8. **导出测试**：导出并在游戏引擎中测试

### 最佳实践

- 先制作基础动画（idle、walk）
- 再制作复杂动画（attack、jump）
- 最后添加细节和优化
- 持续测试和调整
