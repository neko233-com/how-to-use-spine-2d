# 图片与网格

## 导入图片

### 支持的图片格式

- **PNG**：支持透明通道
- **JPG**：不支持透明通道
- **BMP**：位图格式

### 导入步骤

1. 点击 **Images** 面板
2. 点击 **Import** 按钮
3. 选择图片文件
4. 图片会显示在 Images 面板中

### 图片管理

- **重命名**：双击图片名称
- **删除**：选中图片后按 Delete
- **替换**：右键选择 **Replace**

## 绑定图片到骨骼

### 基本绑定

1. 在大纲视图中选中骨骼
2. 在 Images 面板中选中图片
3. 图片会自动绑定到骨骼

### 调整图片位置

1. 选中绑定的图片
2. 使用 Move 工具调整位置
3. 使用 Rotate 工具调整角度
4. 使用 Scale 工具调整大小

### 图片属性

| 属性 | 说明 |
|------|------|
| Position | 位置 (X, Y) |
| Rotation | 旋转角度 |
| Scale | 缩放 (X, Y) |
| Origin | 原点位置 |
| Color | 颜色和透明度 |

## 网格 (Mesh)

### 什么是网格？

网格是将图片分割成多个三角形，可以单独控制每个顶点，实现变形效果。

### 网格优势

- 实现流畅的变形动画
- 减少图片数量
- 更自然的动画效果

### 适用场景

- 角色表情动画
- 布料飘动
- 身体弯曲变形

## 创建网格

### 步骤

1. 选中图片
2. 点击 **Create Mesh** 工具
3. 在图片上点击添加顶点
4. 连接顶点形成三角形
5. 完成网格创建

### 网格编辑

#### 添加顶点

1. 选择 **Edit Mesh** 工具
2. 在网格边缘点击添加顶点
3. 拖动顶点调整位置

#### 删除顶点

1. 选中要删除的顶点
2. 按 Delete 键

#### 调整三角形

- 自动三角剖分
- 手动调整三角形边
- 优化网格结构

### 网格属性

| 属性 | 说明 |
|------|------|
| Hull | 网格外边界 |
| Edges | 网格边 |
| Triangles | 三角形 |
| UV | 纹理坐标 |

## 权重绘制 (Weight Painting)

### 什么是权重？

权重定义了网格顶点受骨骼影响的程度。

### 权重值

- **0.0**：完全不受影响
- **1.0**：完全受影响
- **0.5**：50% 影响

### 绘制权重

1. 选中网格
2. 选择 **Weight** 工具
3. 选择要绘制的骨骼
4. 在顶点上拖动绘制权重

### 权重技巧

- 使用平滑工具过渡权重
- 避免权重突变
- 测试变形效果

## 网格优化

### 减少顶点数

- 删除不必要的顶点
- 合并相近的顶点
- 使用较少的三角形

### 优化三角形

- 避免狭长三角形
- 保持三角形均匀分布
- 手动调整三角形边

### 测试变形

- 在不同姿势下测试网格
- 检查是否有拉伸或撕裂
- 调整权重和网格结构

## 常见问题

### Q: 图片绑定后如何调整位置？

A: 选中图片，使用 Move、Rotate、Scale 工具调整。

### Q: 网格创建后如何修改？

A: 使用 Edit Mesh 工具添加、删除、移动顶点。

### Q: 权重绘制有什么技巧？

A: 使用平滑工具，避免权重突变，多测试变形效果。
