# Installation & Configuration

## System Requirements

### Minimum Requirements

- **Operating System**: Windows 10/11, macOS 10.15+, Linux
- **Processor**: 64-bit processor
- **Memory**: 4 GB RAM
- **Graphics**: OpenGL 3.3 support
- **Storage**: 500 MB available space

### Recommended

- **Memory**: 8 GB RAM or more
- **Graphics**: Dedicated GPU with OpenGL 4.5 support
- **Display**: 1920x1080 or higher resolution

## Download Spine

1. Visit [Spine Official Website](https://esotericsoftware.com/spine-professional)
2. Select the version for your operating system
3. Click the download button

::: warning Note
Spine is paid software, but offers a 30-day free trial.
:::

## Installation Steps

### Windows

1. Run the downloaded installer
2. Follow the installation wizard
3. Launch Spine

### macOS

1. Open the downloaded .dmg file
2. Drag Spine to the Applications folder
3. Launch Spine from Applications

### Linux

1. Extract the downloaded archive
2. Run the `spine` executable
3. Or install via package manager (if available)

## License Activation

1. Launch Spine
2. Click **Help** > **Enter License Key**
3. Enter your license key
4. Click **Activate**

## Preferences Configuration

### Interface Language

1. Click **Edit** > **Preferences**
2. Select language from **Language** dropdown
3. Restart Spine to apply changes

### Keyboard Shortcuts

Spine supports custom keyboard shortcuts:

1. Click **Edit** > **Preferences** > **Shortcuts**
2. Select the command to modify
3. Press the new key combination

### Auto Save

It's recommended to enable auto save:

1. Click **Edit** > **Preferences**
2. Set **Auto Save** interval (recommended: 5 minutes)
3. Set number of auto save files

## FAQ

### Q: Spine won't start, what should I do?

A: Check if your graphics driver is updated and ensure it supports OpenGL 3.3 or higher.

### Q: How do I update Spine?

A: Spine automatically checks for updates, or manually check via **Help** > **Check for Updates**.

### Q: What happens after the trial period ends?

A: Purchase a license or contact Esoteric Software for an extension.
